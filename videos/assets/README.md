# Assets para Vídeos

## Organização

```
assets/
├── minecraft/         # Mundos e construções
├── slides/           # Apresentações
├── diagrams/         # Diagramas e esquemas
├── audio/            # Músicas e efeitos sonoros
└── thumbnails/       # Miniaturas dos vídeos
```

## Padrões de Design

- Principal: #2E2E2E

### Fontes
- Títulos: Roboto Bold
- Corpo: Open Sans
- Código: JetBrains Mono

### Elementos Minecraft
- Texturas: Default
- Shaders: BSL v8.0
- Resource Pack: Vanilla+
